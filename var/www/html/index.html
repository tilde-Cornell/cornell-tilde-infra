tilde@Cornell

A student-run Unix shell server and personal web hosting community for Cornell students.

tilde@Cornell provides shell access, personal web pages, and a shared space for students interested in computing, systems, research, and building things on the open web.

Every member receives a Unix account, SSH access, and a personal site at:

https://this-domain/~username

How to Join

To request an account, connect through SSH and complete the onboarding form:

ssh join@this-domain
Password: GO BIG RED

Applications are reviewed manually. Approved users receive personal shell access and web space.

Important: the application requires your SSH public key (not your private key). Your account cannot be approved without it.

Before applying, make sure you know how to find your existing SSH key or generate a new one.

View SSH Public Key Setup Instructions
